# AI 知识库 · 项目愿景 v1.0

## 要做什么
- 每天抓取 GitHub Trending 20条
- 只抓取 AI 相关项目（topics 包含 ai/llm/agent）
- 用 Agent 分析内容，分析维度包括：项目简介、技术类别、创新点、使用难度、使用场景
- 输出 Markdown 格式的知识条目，字段包括：项目简介、技术类别、创新点、使用难度、使用场景

## 不做什么
- 不做用户系统和权限管理
- 不做 Web 界面（初期仅命令行/脚本运行）
- 不做复杂的历史数据存储和搜索（初期仅保存每日结果）
- 不做实时推送和通知功能
- 不做项目的代码分析和调试功能

## 边界 & 验收
### 边界
- 仅处理 GitHub Trending 上的公开项目
- 每日运行一次，不做实时监控
- 仅分析项目 README 和基本信息，不深入分析代码库
- 使用公开的 GitHub API 和 LLM API

### 验收标准
- 能成功从 GitHub Trending 抓取 20 条 AI 相关项目
- 能为每个项目生成符合要求的 Markdown 知识条目
- 每个条目的分析内容完整、准确、有价值
- 系统能稳定运行，无崩溃或严重错误

## 怎么验证
- 每日运行后，手动抽查 3-5 个项目的分析结果
- 验证条目格式是否符合要求
- 验证分析内容是否准确、有意义
- 验证是否成功抓取了 20 条 AI 相关项目
