# arXiv 中文摘要撰写 Skill

## 执行步骤

1. **读取论文数据** — 读取前序环节生成的经过领域过滤的论文信息
   - 完成标志：成功读取论文数据

2. **理解论文内容** — 深入理解英文标题、摘要和关键词
   - 完成标志：论文内容理解完成

3. **撰写中文摘要** — 为每篇论文撰写精炼的中文摘要
   - 完成标志：所有论文都有中文摘要

4. **摘要质量控制** — 确保中文摘要准确、简洁、易懂
   - 完成标志：摘要质量检查通过

5. **格式统一** — 保持所有论文摘要格式的一致性
   - 完成标志：格式统一检查通过

6. **输出** — 按以下 JSON 格式返回结果
   - 完成标志：输出已格式化并通过 JSON schema 校验

## 中文摘要撰写规范

### 摘要结构
严格遵循「研究主题 + 核心方法 + 主要贡献」的三段式结构：
1. **研究主题** — 一句话说明论文要解决的问题
2. **核心方法** — 简要介绍采用的技术或方法
3. **主要贡献** — 概括论文的创新点和价值

### 质量要求
- 字数控制在 50-150 字之间
- 语言准确、流畅、专业
- 避免直译英文术语，使用中文领域通用表达
- 突出论文的创新点和实用价值
- 确保不添加原文未包含的信息

```json
  }
]
```

- 中文摘要严格遵循三段式结构
- 字数控制在 50-150 字之间
