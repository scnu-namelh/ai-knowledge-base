# arXiv 论文详情采集 Skill

## 执行步骤

1. **读取论文列表** — 读取前序环节生成的论文基本信息列表（JSON 格式）
   - 完成标志：成功读取论文列表数据

2. **访问详情页面** — 逐篇访问 arXiv 论文详情页面
   - 完成标志：每篇论文的详情页面成功访问

3. **提取完整摘要** — 获取论文的完整英文摘要
   - 完成标志：摘要完整提取，无截断

4. **收集附加信息** — 提取论文评论、会议信息、期刊引用等附加元数据
   - 完成标志：所有可用的附加信息已提取

5. **批量处理** — 高效处理多篇论文，保持数据完整性
   - 完成标志：所有论文均完成详情采集

6. **输出** — 按以下 JSON 格式返回结果
   - 完成标志：输出已格式化并通过 JSON schema 校验

## 输出格式

```json
[
  {
    "arxiv_id": "2607.08758",
    "title": "论文标题",
    "authors": ["作者1", "作者2"],
    "url": "https://arxiv.org/abs/2607.08758",
    "pdf_url": "https://arxiv.org/pdf/2607.08758",
    "abstract": "完整英文摘要内容",
    "comments": "论文评论（如适用）",
    "submitted_date": "2026-07-09",
    "categories": ["cs.AI", "cs.LG"],
    "source": "arxiv"
  }
]
```

- 保留所有原始字段
- 添加完整英文摘要
- 如有评论信息也一并提取
- 处理失败的论文需保留错误记录
